MDC SideStore Warning
The MDC version of SideStore above (bypass 3-app limit) is only compatible with iOS 15.0-16.1.2, and is very outdated, and so you may experience issues. If you want to have this feature, we reccomend you join our Discord and go to the sidestore-downloads channel, which will contain MDC compatible SideStore builds. Then, seperately install WDBRemoveThreeAppLimit with SideStore to apply the patch.
